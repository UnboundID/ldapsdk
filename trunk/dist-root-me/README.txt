This package contains the free edition of the UnboundID LDAP SDK for Java in a
minimal form intended for space-constrained environments, like embedded
applications (e.g., on Android systems) in which space is at an absolute
premium.  For environments without such tight constraints, it is recommended
that the Standard Edition of the LDAP SDK be used instead.

The following elements have been provided:

- unboundid-ldapsdk-me.jar -- The UnboundID LDAP SDK for Java (Minimal
  Edition) library.

- docs/index.html -- A list of available documentation.

- LICENSE.txt -- Provides information about the licenses under which the
  UnboundID LDAP SDK for Java (Minimal Edition) is made available.

- LICENSE-GPLv2.txt -- The license text for the GPLv2 license under which the
  UnboundID LDAP SDK for Java (Minimal Edition) may be used.

- LICENSE-LGPLv2.1.txt -- The license text for the LGPLv2.1 license under which
  the UnboundID LDAP SDK for Java (Minimal Edition) may be used.

- LICENSE-UnboundID-LDAPSDK.txt -- The license text for the UnboundID-specific
  free right to use license under which the UnboundID LDAP SDK for Java
  (Minimal Edition) may be used.
