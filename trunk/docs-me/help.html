Getting Help

The home page for the UnboundID LDAP SDK for Java is available online at ${LDAP_SDK_HOME_URL}, and the documentation is available at ${LDAP_SDK_DOCS_BASE_URL}. The documentation is also included in the docs directory of the LDAP SDK product download.

If you have questions, comments, problem reports, or other feedback regarding the LDAP SDK, then you may provide it using one of the following options:

  • Use the online discussion forum, available at https://sourceforge.net/projects/ldap-sdk/forums/forum/1001257.

  • Send an e-mail message to ldapsdk-support@unboundid.com or ldap-sdk-discuss@lists.sourceforge.net.

  • Purchase a commercial support contract for the LDAP SDK. More information about obtaining a support contract may be found on the LDAP SDK product home page.
